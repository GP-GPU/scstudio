Shape Selection

On top of standard Visio functions for selecting shapes (such as Ctrl+A shortcut for selecting all shapes), SCStudio offers several useful selection functions.

The first toolbar button selects all instances in the current page of active document. The second button selects all messages in the current page of active document. These functions are also available via menu Check → Drawing → Select → All Instances, or All Messages, respectively. Keyboard shortcuts are assigned to these functions, too: Ctrl+Alt+I for selecting all instances and Ctrl+Alt+M for selecting all messages.

While holding Ctrl or Shift, the first two buttons alter their face and functionality, however: In that case, they add all instances (or messages) to the current selection instead of making a new selection.
